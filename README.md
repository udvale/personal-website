# Personal Website
Portfolio Website Made with React.JS and CSS.

## Installation & Set Up

1. Install the Gatsby CLI

   ```sh
   npm install -g gatsby-cli
   ```

2. Install dependencies

   ```sh
   yarn
   ```

3. Start the development server

   ```sh
   npm start
   ```

## Building Production

1. Generate production build

   ```sh
   npm run build
   ```
